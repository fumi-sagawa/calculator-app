# calcurator-app

シンプルな電卓です。(開発中)
https://calculator-app-gamma.vercel.app/

## 構成

- Next.js
- Emotion
- Redux
- Redux-Toolkit
- TypeScript
- ESLint
- Prettier
- husky
- lint-staged

## 使い方

```
インストール
yarn

起動
yarn dev

ビルド
yarn build
yarn start
```
